# Holi!
Este proyecto consiste en el diseño y creación de mi portfolio personal, realizado en HTML para la materia Computación III de la Lic. en Diseño y Comunicación Visual.
La consigna consistía en crear una página web personal, aplicar estilos mediante CSS, desarrollar el maquetado previo en Figma e implementar recursos como Media Queries para diseño responsive, animaciones CSS y galerías de imágenes mediante FancyBox.
## Detalles técnicos
* Mi pantalla tiene una resolución de 1920x1080px.
* El editor de código que usé es Visual Studio Code.
* El prototipo y maquetado fueron realizados en Figma, utilizando herramientas como Auto Layout y prototipado interactivo.
